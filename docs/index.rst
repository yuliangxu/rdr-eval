RDR Eval
========

RDR Eval is a lightweight PyTorch package for estimating the relative density
ratio

.. math::

   r(x) = \frac{2p(x)}{p(x)+q(x)}.

The package provides bounded neural ratio estimation, multiple divergence
objectives, reproducible train/validation/test splitting, and validation-based
early stopping.
